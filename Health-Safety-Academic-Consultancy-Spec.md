# Health & Safety & Academic Consultancy Website Specification

## 1. Project Overview

The objective of this project is to build a modern, professional, responsive, secure, and user-friendly website for a consultancy platform offering Health & Safety and Academic Writing services.

The website will serve three major purposes:

- Publish free and premium educational articles.
- Allow users to unlock premium content through secure online payments.
- Enable visitors to hire professional consultancy services directly from the website.

## 2. Design Direction

The website should feel premium, international, trustworthy, and conversion-focused.

### Visual Style

- Modern and minimal
- Professional and credible
- Clean visual hierarchy
- Fast and responsive
- Strong typography and generous spacing
- Premium consultancy tone, not a typical blog aesthetic

### Typography

- The entire website must use the provided font file: public/Rency-Regular.woff2
- This font should be applied consistently across all pages and components

## 3. Main Pages

### Home Page

The Home Page should include:

- Hero Banner
- Brief Introduction
- Services Overview
- Featured Articles
- Premium Articles
- Why Choose Us
- Client Testimonials
- Latest Blog Posts
- Frequently Asked Questions (FAQs)
- Call-to-Action (CTA) Buttons

- Contact Section

### Academic Consultancy Page

The Academic Consultancy section should include:

- Undergraduate Projects
- Master’s Thesis
- PhD Proposal
- Dissertation Writing
- Research Methodology
- Literature Review
- Data Analysis
- SPSS Analysis
- STATA Analysis
- Excel Analysis
- Questionnaire Design

- Proofreading

### Professional Writing Page

The Professional Writing section should include:

- Personal Statements
- Statement of Purpose (SOP)
- Grant Proposals
- Business Proposals
- Business Plans
- Technical Reports

## 4. Blog System

The blog will be a major feature of the website.

### Admin Capabilities

The administrator should be able to:

- Create Posts
- Edit Posts
- Delete Posts
- Upload Images
- Upload PDF Attachments
- Add Categories
- Add Tags
- Schedule Posts
- Feature Important Posts

### Blog Experience

The blog should support:

- Search
- Categories
- Tags
- Featured posts
- Recent posts
- Popular posts
- Related articles
- Author information
- Breadcrumb navigation
- Table of contents for long posts
- Reading time display
- Share buttons
- Newsletter signup
- Pagination

## 5. Premium Locked Posts

This is one of the core functionalities of the platform.

### Free Articles

Some articles should be available for free.

### Premium Articles

Some articles should require payment before full access.

Premium articles should display the message:

> This is Premium Content.
>
> Unlock this article by making payment.

Only a preview should be visible before payment.

### After Successful Payment

- The article unlocks immediately.
- The user gains instant access.
- An email confirmation is sent.
- The purchased article is permanently available inside the user’s account.

## 6. Payment System

Integrate secure payment gateways.

### Preferred Payment Gateways

- Stripe
- Paystack
- Flutterwave
- PayPal

### After Successful Payment

The system should:

- Automatically unlock purchased content.
- Send a payment receipt to the customer.
- Notify the administrator.
- Record payment history.

## 7. User Registration and Account

Visitors should be able to:

- Register
- Login
- Reset Password
- Manage Profile
- View Purchased Articles
- View Payment History
- Download Purchased Files (where applicable)

Users should also have the option to continue as guests where registration is not required.

### Account Pages

- Dashboard
- Profile
- Purchased Articles
- Payment History
- Downloads
- Settings

## 8. Booking System

Visitors should be able to book professional consulting services.

### Booking Categories

- NEBOSH Writer
- Health & Safety Consultant
- Academic Consultant
- Research Assistant
- Project Supervisor
- Dissertation Consultant
- Statistical Data Analyst
- Risk Assessment Specialist
- HSE Documentation Consultant

### Booking Form Fields

The booking form should capture:

- Full Name
- Email Address
- Phone Number
- WhatsApp Number
- Country
- Service Required
- Project Title
- Deadline
- Budget (Optional)
- Project Description
- File Upload

### Booking Outcome

After submission:

- The administrator should receive an instant notification.
- A confirmation message should be shown to the user.

## 9. File Upload

Clients should be able to upload the following file types:

- Microsoft Word Documents (.doc/.docx)
- PDF Files
- Microsoft Excel Files (.xls/.xlsx)
- Microsoft PowerPoint Files (.ppt/.pptx)
- Images
- ZIP Archives

### Upload Limit

- Maximum file size: 100 MB

The website should provide:

- SEO-friendly

The website should provide a complete consultancy platform that combines:

- Professional consultancy services
- Academic writing services
- Professional writing services
